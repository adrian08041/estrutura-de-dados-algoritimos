# 👨‍💻 Estrutura de Dados e Algoritmos

Este repositório contém implementações e exercícios desenvolvidos durante a disciplina de **Estrutura de Dados e Algoritmos**, ministrada no curso de Sistema de Informação pela UNIPAM.

## 📚 Objetivo

O objetivo deste projeto é consolidar os principais conceitos estudados na matéria, como:

- Estruturas de dados lineares (listas, pilhas, filas)
- Estruturas de dados não lineares (árvores, grafos)
- Algoritmos de ordenação e busca
- Análise de complexidade
- Recursividade
- Algoritmos gulosos, divisão e conquista, programação dinâmica, entre outros
